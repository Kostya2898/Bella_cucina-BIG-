⚙️ Як налаштувати

Встановіть змінні середовища перед запуском сервера:

# Windows (PowerShell)
$env:MAIL_USERNAME = "your@gmail.com"
$env:MAIL_PASSWORD = "your_app_password"
$env:MAIL_DEFAULT_SENDER = "Bella Cucina <your@gmail.com>"

# Linux / Mac
export MAIL_USERNAME="your@gmail.com"
export MAIL_PASSWORD="your_app_password"
export MAIL_DEFAULT_SENDER="Bella Cucina <your@gmail.com>"

💡 Gmail: увімкніть двоетапну перевірку → згенеруйте App Password (не звичайний пароль!) на myaccount.google.com/apppasswords
